# Лабораторна робота №7
## Тема

Обробка IO/мережевих помилок та патерн Retry в C#

## Мета роботи

Навчитися обробляти типові помилки вводу/виводу та мережеві помилки за допомогою блоків try-catch-finally, а також реалізувати патерн Retry для підвищення відмовостійкості застосунків.

## Теоретичні відомості

* **Винятки вводу/виводу**: FileNotFoundException, IOException.
* **Мережеві винятки**: HttpRequestException.
* **try-catch-finally**: базовий механізм обробки винятків у C#.
* **Патерн Retry**: повторне виконання операції при тимчасових помилках.
* **Експоненційна затримка**: збільшення часу очікування між спробами.

## Хід роботи

1. Створено консольний проєкт `lab7vN`.
2. Реалізовано клас `RetryHelper` з методом `ExecuteWithRetry` для повторних спроб з експоненційною затримкою.
3. Створено клас `FileProcessor` для імітації помилок при роботі з файлами.
4. Створено клас `NetworkClient` для імітації мережевих помилок.
5. У `Main` продемонстровано роботу `RetryHelper`:

   * Виклик методів `ReadFile` та `DownloadData`.
   * Логування кожної спроби та помилок.
   * Використання делегата `shouldRetry` для вибіркової обробки винятків.

## Результати роботи

* Повторні спроби виконуються при помилках `FileNotFoundException` та `HttpRequestException`.
* Після кількох невдалих спроб операції успішно виконуються.
* В консолі відображаються номери спроб, типи помилок та час очікування.

## Висновки

* Опановано роботу з IO- та мережевими винятками.
* Реалізовано патерн Retry з експоненційною затримкою.
* Продемонстровано відмовостійкість застосунку при тимчасових збоях.
* Мета роботи досягнута.
